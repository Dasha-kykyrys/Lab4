# Shape Manager Project

Проект для управления геометрическими фигурами с консольным интерфейсом.

## Описание

Программа позволяет:
- Создавать различные геометрические фигуры (круг, прямоугольник, квадрат, треугольник)
- Управлять коллекцией фигур (добавление, удаление, просмотр)
- Вычислять статистические данные (общая площадь, периметр, средняя площадь)
- Выполнять поиск фигур по различным критериям
- Сохранять и загружать отчеты в формате JSON


## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Скопируйте все файлы проекта в одну директорию
3. Запустите программу:

```bash
python console_ui.py

# Задача: Классы Circle и Rectangle

**Ответственный:** Даша
**Статус:** 
**Срок:** 

## Описание
Реализовать конкретные классы для круга и прямоугольника с валидацией данных

## Требования
### Circle
- [ ] Создать класс `Circle` в файлах `Circle.h` и `Circle.cpp`
- [ ] Наследование от `Shape`
- [ ] Приватное поле `radius` (double)
- [ ] Реализация `calculateArea()` (π*r²)
- [ ] Реализация `calculatePerimeter()` (2*π*r)
- [ ] Валидация: радиус должен быть > 0
- [ ] Конструктор с параметрами: `Circle(double radius, string color)`

### Rectangle
- [ ] Создать класс `Rectangle` в файлах `Rectangle.h` и `Rectangle.cpp`
- [ ] Наследование от `Shape`
- [ ] Приватные поля: `width` и `height` (double)
- [ ] Реализация `calculateArea()` (width*height)
- [ ] Реализация `calculatePerimeter()` (2*(width+height))
- [ ] Метод `bool isSquare() const` для проверки на квадрат
- [ ] Валидация: width и height должны быть > 0
- [ ] Конструктор: `Rectangle(double w, double h, string color)`

## Описание
Реализовать конкретные классы для круга и прямоугольника

## Требования
### Circle
- [ ] Наследование от Shape
- [ ] Поле radius
- [ ] Реализация calculateArea() и calculatePerimeter()
- [ ] Валидация радиуса (> 0)

### Rectangle
- [ ] Наследование от Shape
- [ ] Поля width и height
- [ ] Метод isSquare() для проверки на квадрат
- [ ] Валидация сторон (> 0)
